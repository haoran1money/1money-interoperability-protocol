## emerald

This repository includes emerald as a Git submodule.

If you cloned the repository without submodules, initialize them with:

```
git submodule update --init --recursive
```


To clone the repository with all submodules initialized:

```
git clone --recurse-submodules git@github.com:haoran1money/1money-interoperability-protocol.git
```
